添加数字蚁后
不论蚁后的状态是哪一种，都可以通过系统进行赠送。
每小时的产量 等于 预计收入 除以 运行周期。
如果有小数，则保留8位小数，假如第9位小数大于等于5，那么则四舍五入到第8位。
举例：A蚁后，周期7小时，产量10货币，前6个小时的产量为1.42857143，最后一个小时的产量为1.42857142。
